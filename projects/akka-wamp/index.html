Akka Wamp

Akka Wamp is a WAMP - Web Application Messaging Protocol implementation written for both Scala and Java developers that wish to use Akka abstractions to write concurrent and reactive applications that are easy to reason about.

It provides you with

  • object-oriented representations of WAMP messages,
  • intuitive Client APIs designed to be used with actors, futures and streams,
  • basic Router you can embed in your server application or launch as standalone.

Usage

Make your Scala or Java build depend on the latest version of akka-wamp:

sbt
libraryDependencies ++= Seq(
  "com.github.angiolep" % "akka-wamp_2.11" % "0.13.0"
  // ...
)
mvn
<dependencies>
    <dependency>
        <groupId>com.github.angiolep</groupId>
        <artifactId>akka-wamp_2.11</artifactId>
        <version>0.13.0</version>
    </dependency>
    <!-- .... -->
</dependencies>
gradle
dependencies {
  compile 'com.github.angiolep:akka-wamp_2.11:0.13.0'
  // ...
}

Then read on Client APIs for further details.

Requirements

  • Java >= 1.8
  • Scala >= 2.11

Limitations

  • WebSocket transport only (no raw TCP and no SSL/TLS yet)
  • WAMP Basic Profile only (no Advanced Profile yet)
  • JSON serialization only (no MsgPack yet)
